Ripple

Ripple is a real-time gross settlement system (RTGS), currency exchange and remittance network. It is built upon a distributed open source internet protocol, consensus ledger and native cryptocurrency called XRP (ripples).

It purports to enable secure instantly and nearly free global financial transactions of any size with no chargebacks.

Ripple is based around a shared public database/ledger. This uses a consensus process that allows for payments exchanges and remittance in a distributed process.

I believe Ripple has the potential to play a key part in evolving banking in this digital era.
